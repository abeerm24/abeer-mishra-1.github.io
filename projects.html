This page enlists and briefly describes various projects that I have completed followed by the attached Google collab notebook or the code. I have tried to explain the code everywhere as much as possible. I will be adding more projects over here over time.

PH 421: Photonics group project - A non linear Kerr optical polarization controller

In this project, we explain the paper on "A Kerr Polarization Controller" by N. Moroney,L. Del Bino, et al. We prepared an interactive, executable paper on Google collab that can be used to understand, visualize and obtain the results of the paper. We explain the math, theoretically replicated the graphs, provided interactive toy models to understand the concepts involved and obtained the raw data from the authors and used it to obtain the experimental plots in the paper. Single Image super-resolution using CNNs - ESPCNN

This project contains an implementation of the efficient sub-pixel convolutional neural network (ESPCNN) for single-image or single-frame video super-resolution. The model is implemented using the Pytorch library in Python. It was done as a part of a group effort to study various CNN-based super-resolution techniques developed till today. The results can be further improved by more extensive training.


EE 610: Image processing - Wavelet transform based image denoising

In this project, we compare 3 Wavelet transform based denoising techniques for color images - Hard thresholding, Soft thresholding, and a method given by Chen et al in their paper on "Image denoising using neighbouring wavelet coefficients". We statistically study the impact of adding noise and the relative performance of the 3 methods on denoising the color images.


EE 610: Image processing - CNN based nuclei segmentation

In this project, we build a UNet model using Keras and Tensorflow and train it on a relatively small dataset of tissue images. We use the MoNuSeg Challenge training dataset to train our UNet model. Finally we implement the watershed algorithm to segment the nuclie from the background. The results can be enhanced by further training to improve the segmentation results.


PH 435: Microprocessors - Piano tiles on Arduino

In this project, we recreated the classic Piano tiles game using an Arduino Uno, a 2.4 inch TFT LCD touchscreen and a piezobuzzer. We make use of "structs" to randomly initialize and keep track of the piano tiles. We adjusted the game speed as per Arduino's processing power in order to reduce lag time for best user experience. SC 649: Embedded control and Robotics - Trilateration algorithm in ROS

This project contains an implementation of the trilateration algorithm with orientation of the robot assumed to be known. The robot is then simulated in the Gazebo plugin for ROS.
